# Игра "Змейка"

## Как поиграть?
1. Установить [JVM](https://www.java.com/ru/download/).

2. Скачайть `.jar`-файл из раздела релизов.

3. Запустить игру командой:

```shell
java -jar .\Snake.jar
```

**Для управления используются клавиши WASD**
